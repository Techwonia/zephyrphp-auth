# ZephyrPHP Auth

Authentication module for [ZephyrPHP](https://zephyrphp.com) — session and JWT authentication.

## Features

- Session-based authentication
- JWT token generation & validation (HS256)
- Remember me tokens
- User provider with Doctrine ORM
- Auto-detects User model namespace

## Installation

```bash
composer require zephyrphp/auth
```

> **Note:** This package is built for ZephyrPHP and is not compatible with Laravel or other PHP frameworks.
